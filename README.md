# Livestock Record Management System

A PHP & MySQL project for managing livestock records, including animals, vaccination, milk, breeding, and sales.

## Features
- Add, view, edit, and delete animal records
- Vaccination management
- Milk production tracking
- Breeding management
- Sales tracking and reporting
- Role-based access for admin and users
- Professional UI with Bootstrap dashboard

## Technologies Used
- PHP
- MySQL
